# 2022 개정 교육과정 교수·학습 과정안(약안) 자동 생성

연간지도 계획, 성취기준·성취수준, 핵심아이디어, 장학자료를 반영해 **약안 서식**으로 과정안을 자동 생성하는 웹앱입니다.

## ⚠️ API 키는 저장소에 없습니다

- **`.env` 파일은 git에 올라가지 않습니다.** (`.gitignore`에 포함)
- 과정안 생성(Gemini API)을 쓰려면 **로컬에서만** `.env`를 만들어야 합니다.

### API 키 설정 (로컬에서만)

1. [Google AI Studio](https://aistudio.google.com/apikey)에서 Gemini API 키 발급
2. 프로젝트 폴더에 `.env` 파일 생성 후 아래 한 줄 추가:
   ```env
   GEMINI_API_KEY=발급받은_키_입력
   ```
3. 또는 `.env.example`을 복사해 `.env`로 이름만 바꾼 뒤, `your_api_key_here`를 실제 키로 수정

## 실행 방법

```bash
npm install
npm start
```

브라우저에서 **http://localhost:3000** 으로 접속한 뒤, 학년·교과·단원·차시를 선택하고 **과정안 생성**을 누르면 됩니다.

## 포함된 자료

- 연간지도 계획 (`연간지도_계획.json`)
- 성취기준 및 해설, 성취수준, 핵심아이디어, 적용 시 고려사항, 장학자료 등 (프로젝트 내 JSON·txt)
