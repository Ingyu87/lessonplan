# 연간지도 계획 데이터 형식

`연간지도_계획.json` 파일에 교과별·학년별 단원 정보를 추가하면, AI가 해당 자료를 바탕으로 과정안을 작성합니다.

## JSON 형식

```json
[
  {
    "교과": "국어",
    "학년": "3",
    "학년군": "3~4학년",
    "단원목록": [
      {
        "단원번호": 1,
        "단원명": "바른 말을 사용해요",
        "차시수": 9,
        "주요_학습_내용_및_활동": "인사말과 바른 말 사용하기, 말차례 지키며 대화하기, 경험 나누기"
      }
    ]
  }
]
```

- **교과**: 국어, 수학, 사회, 과학 등
- **학년**: 1~6
- **학년군**: 1~2학년, 3~4학년, 5~6학년
- **단원목록**: 해당 교과·학년의 단원 배열
  - **단원번호**: 단원 순서
  - **단원명**: 단원 이름 (성취기준 JSON의 단원명과 일치하면 좋음)
  - **차시수**: 해당 단원의 총 차시 수
  - **주요_학습_내용_및_활동**: 해당 단원의 주요 학습 내용 및 활동 (AI 프롬프트에 반영)

## 장학자료

`장학자료` 폴더를 만들고, 교과명이 포함된 파일을 넣으면 AI가 참고합니다.

- 예: `장학자료/국어_3학년.txt`, `장학자료/수학_지도자료.json`
- 파일명에 교과명(국어, 수학 등)이 포함되어야 합니다.
