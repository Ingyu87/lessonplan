# PDF 분석 → 자료 생성

## 실행 방법

```bash
# 모든 PDF 분석 후 자료 JSON 생성
npm run pdf
# 또는
node pdf-all-to-data.js
```

## 생성되는 자료

| 파일 | 출처 | 설명 |
|------|------|------|
| `연간지도_계획.json` | 2026년도 연간 지도 계획 PDF 44개 | 교과·학년별 단원, 차시수, 주요 학습 내용 및 활동 |
| `약안_서식_구조.json` | 2022 개정 교수·학습 과정안(약안 서식).pdf | 약안 서식 섹션 구조, 평가범주, 유의사항 |
| `자료_인덱스.json` | - | 전체 자료 목록 및 메타정보 |

## 기존 JSON 자료 (PDF 아님)

- `2022개정교육과정 성취기준 및 해설.json`
- `★(초)2022개정교육과정에따른성취수준(*학년군).json` (1~2, 3~4, 5~6학년군)
- `핵심아이디어.txt`, `성취기준 적용시 고려사항.txt`

## 처리 대상 PDF

1. **연간지도 계획**: `2026년도_N학년 N학기_교과_연간 지도 계획.pdf` 형식
2. **약안 서식**: 파일명에 "약안", "교수·학습 과정안", "서식" 포함
